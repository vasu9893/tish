üîß How to Get Your JWT Token

Step 1: Login to Your Dashboard

Go to your dashboard and login:

Step 2: Open Browser Developer Tools

Press F12 or right-click and select "Inspect" to open developer tools.

Step 3: Go to Console Tab

Click on the "Console" tab in the developer tools.

Step 4: Get Your Token

Type this command in the console and press Enter:

localStorage.getItem('token')

This will return your JWT token. Copy the entire token (it's a long string).

Step 5: Use the Token

Paste the token into the test tool and run the tests again.
